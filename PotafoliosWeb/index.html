Gestión de Bovinos

Este es el Proyecto Final con el que conseguí junto a mi compañero Nicolas Troche el título de Analista Programador.
En él se desarrolló un sistema para administrar y gestionar la sanidad y la consanguinidad en bovinos lecheros de forma sencilla, para el Establecimiento La Palma S.R.L.
En cuanto a las herramientas de desarrollo utilizamos JAVA y MySQL para el desarrollo y manejo de base de datos. En cuanto al IDE utilizado, comenzamos utilizando IntelliJ IDEA y en la parte final decidimos cambiar a Apache NetBeans IDE para mayor comodidad a la hora de hacer la interfaz
